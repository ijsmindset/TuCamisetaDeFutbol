# Tienda de Camisetas de Fútbol

Proyecto desarrollado como parte del ciclo de Desarrollo de Aplicaciones Web (DAW). Se trata de una página web estática creada únicamente con HTML y CSS, cuyo objetivo es simular una tienda online de camisetas de fútbol y servir como base para futuros desarrollos más avanzados.

## Aviso legal (Disclaimer)

Este proyecto es exclusivamente académico.  
Las imágenes utilizadas pertenecen a sus respectivos propietarios y se emplean solo con fines educativos.  
Los precios, productos y marcas mostrados son simulaciones y no representan una tienda real.

## Descripción del proyecto

La web está compuesta por varias páginas accesibles desde un menú principal:

- **Inicio**: presentación general de la tienda.
- **Camisetas**: catálogo principal, incluyendo un pequeño vídeo integrado al hacer scroll.
- **Contacto**: formulario básico para simular el envío de consultas.
- **Aviso legal**: página dedicada al disclaimer del proyecto.

El diseño está planteado para ser sencillo, claro y fácil de ampliar, permitiendo añadir lógica de programación o integrar un backend en el futuro.

## Tecnologías utilizadas

- **HTML5** para la estructura del sitio.
- **CSS3** para la maquetación y los estilos.

No se ha utilizado JavaScript ni ningún lenguaje de servidor, ya que el objetivo principal era trabajar la maquetación y la organización de un sitio multipágina.

## Objetivo del proyecto

El propósito de esta web es practicar la maquetación, la estructura de un sitio multipágina y la organización de recursos.  
Además, la estructura está pensada para poder reutilizarse como plantilla en otros nichos (ropa, electrónica, productos digitales, etc.) o para añadir funcionalidad mediante JavaScript o un backend.

## Mejoras previstas

- Añadir interacción con JavaScript.
- Implementar un backend para gestionar productos.
- Crear un sistema real de carrito de compra.
- Mejorar el diseño responsive.
- Integrar una base de datos para almacenar productos y precios.
